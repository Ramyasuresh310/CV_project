# Food Image Recognition System

## Project Overview
This project focuses on building a deep learning-based food image recognition system using computer vision techniques. The objective is to classify food images into different categories using image preprocessing, data augmentation, and CNN-based deep learning models.

## Dataset
- Dataset: Food-101 Dataset
- Selected Categories: 17 food classes
- Data Type: Food images

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab / Jupyter Notebook

## Project Workflow
1. Loaded and explored the food image dataset
2. Performed image size and brightness analysis
3. Checked class distribution and dataset imbalance
4. Applied image preprocessing techniques
5. Used data augmentation to improve model generalization
6. Built CNN-based image classification models
7. Trained and evaluated the model
8. Analyzed model performance using visualizations
